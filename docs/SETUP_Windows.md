# Windows ではじめての起動ガイド（コピペでOK）

ターミナルがほぼ初めての方向けに、**最初から最後まで**書いた手順です。
ゴールは「ブラウザで `http://localhost:5173` を開いたらダッシュボードが見える」状態。

所要時間：初回は 20〜30分くらい（ダウンロード待ちが大半）。

---

## 全体像（4ステップ）

```
① 道具を2つ入れる（Python と Node.js）   ← 1回だけ
② コードを自分のPCにダウンロード          ← 1回だけ
③ start-backend.bat と start-frontend.bat をダブルクリック
④ ブラウザで http://localhost:5173 を開く  → 完成！
```

---

## ① 道具を2つインストールする（1回だけ）

### Python（バックエンドを動かす言語）

1. https://www.python.org/downloads/ を開く
2. 黄色い「Download Python 3.x.x」ボタンを押してインストーラを保存
3. ダウンロードした `.exe` を実行
4. **【超重要】最初の画面で下にある「Add python.exe to PATH」に必ずチェック** ✅
   （これを忘れると動きません）
5. 「Install Now」を押して、完了するまで待つ

### Node.js（画面・フロントエンドを動かす道具）

1. https://nodejs.org/ を開く
2. 「LTS」と書かれた方のボタンを押してインストーラを保存
3. ダウンロードした `.msi` を実行
4. ほぼ「Next」を押し続けるだけでOK（設定変更不要）

> インストール後、念のため **パソコンを一度再起動**すると確実です。

---

## ② コードを自分のPCにダウンロードする（1回だけ）

一番かんたんな「ZIPダウンロード」方式：

1. ブラウザで GitHub のこのリポジトリを開く
   `https://github.com/purplecom24-netizen/kuroshiba-game`
2. 画面上部のブランチ切替（"main" と書かれたボタン）を押し、
   **`claude/semi-auto-trading-tool-47nifp`** を選ぶ
   （ここに最新のコードが入っています）
3. 緑色の **「Code」ボタン → 「Download ZIP」** を押す
4. ダウンロードした ZIP を**右クリック →「すべて展開」**
5. 展開先は分かりやすい場所（例：デスクトップ）にする

展開すると `kuroshiba-game` というフォルダができ、中に
`start-backend.bat` と `start-frontend.bat` が入っています。

---

## ③ 起動する（毎回これだけ）

ターミナルにコマンドを打つ必要はありません。**ダブルクリックするだけ**です。

1. `start-backend.bat` を**ダブルクリック**
   - 黒い画面が開き、初回は自動でセットアップが走ります（数分）
   - `Application startup complete` のような表示が出たら成功
   - **この黒い画面は閉じないでください**（閉じると止まります）

2. 次に `start-frontend.bat` を**ダブルクリック**
   - もう1つ黒い画面が開き、初回は自動セットアップ（数分）
   - `Local: http://localhost:5173/` と表示されたら成功
   - **こちらの黒い画面も開いたままに**

> 黒い画面が一瞬で消える / エラーが出る場合は、画面に書かれた
> メッセージ（Python や Node が見つからない 等）を読んでください。
> ①のインストールやPATHのチェック漏れが原因のことが多いです。

---

## ④ ブラウザで開く

ブラウザ（Chrome等）のアドレス欄に次を入力してEnter：

```
http://localhost:5173
```

ダーク画面のダッシュボードが表示され、左の銘柄リストをクリックすると
ローソク足チャートが出れば成功です 🎉
（この時点ではキー未設定なので「モックデータ」の練習用チャートです）

---

## 止め方

2つの黒い画面それぞれで **Ctrl + C** を押すか、画面を閉じればOKです。

---

## 次のステップ（任意）：本物の株価にする

Alpaca のペーパー用キーを入れると、練習用データが本物の株価に変わります。

1. `backend` フォルダの中の `.env.example` をコピーして、名前を `.env` に変更
2. メモ帳で `.env` を開き、`ALPACA_PAPER_KEY_ID=` と
   `ALPACA_PAPER_SECRET_KEY=` の右に Alpaca で取得したキーを貼る
3. バックエンドの黒い画面を一度閉じて、`start-backend.bat` を再ダブルクリック

ステータスバーの `data: mock` が `data: alpaca` に変われば成功です。

> ⚠️ `.env` は秘密のファイルです。GitHub にアップロードしたり、
> 人に見せたりしないでください（自動で除外設定済みです）。
